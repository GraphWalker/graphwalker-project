GraphWalker Studio
==================

## Build Studio

```bash
mvn package
```

The jar is in:
```bash
target/graphwalker-studio-4.0.0-SNAPSHOT.jar
```

Run it like:
```bash
java -jar target/graphwalker-studio-4.0.0-SNAPSHOT.jar
```
